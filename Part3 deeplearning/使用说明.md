使用说明
安装依赖：

bash
cd Part1_WAFFingerprint
pip install -r requirements.txt
训练模型（如果需要）：

python
from lib.ml_classifier import WAFClassifier
from lib.utils import load_training_data

# 加载训练数据
features, labels = load_training_data('data/training_data.csv')

# 训练模型
classifier = WAFClassifier('random_forest')
result = classifier.train(features, labels)
classifier.save_model('models/waf_classifier.pkl')
使用检测工具：

bash
# 被动检测
python bin/start_detection.py https://example.com -m passive

# 主动检测  
python bin/start_detection.py https://example.com -m active

# 综合检测
python bin/start_detection.py https://example.com -m comprehensive -o result.json
这个WAF指纹识别系统结合了机器学习和规则检测，能够准确识别多种主流WAF产品，为渗透测试和安全评估提供有力支持。
